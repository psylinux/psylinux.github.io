Packer VM Templates

Date: 2018-12-16
Reading Time: 1 min
Section: Portfolio
Categories: portfolio
Tags: vm-packer

Packer VM Templates :: is a collection of Packer.io templates used to automate the creation of Virtual Machine images.

Contain:

  • Kali Linux Template: A specific configuration (packer-kali-template) for building Kali Linux images.
  • Generic/Base Templates: Other general templates (packer-template). It is open-source (BSD 3-Clause License) and allows for reproducible builds of these VM environments as code.
