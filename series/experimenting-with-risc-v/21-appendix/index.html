References and further reading

  • RISC-V ISA Manual (Volume I: Unprivileged): https://github.com/riscv/riscv-isa-manual/releases
  • RISC-V ELF psABI: https://github.com/riscv-non-isa/riscv-elf-psabi-doc
  • QEMU RISC-V system emulation: https://www.qemu.org/docs/master/system/target-riscv.html
  • GCC RISC-V options: https://gcc.gnu.org/onlinedocs/gcc/RISC-V-Options.html
  • GDB manual: https://sourceware.org/gdb/current/onlinedocs/gdb/

Additional resources

  • RISC-V Assembly Programmer’s Manual: https://github.com/riscv-non-isa/riscv-asm-manual
  • GNU binutils manuals (objdump, readelf, objcopy): https://sourceware.org/binutils/docs/
  • QEMU virt machine documentation: https://www.qemu.org/docs/master/system/riscv/virt.html

Exercises

  1. Open the RISC-V ISA manual and find the encoding for add.\n2. Locate the QEMU virt memory map and list the UART base address.

Summary

These references are the authoritative sources for instruction behavior, ABI rules, and emulator options. Use them when you need exact specifications or edge-case details.
